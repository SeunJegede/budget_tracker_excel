# Personal Budget Tracker
A Excel/Google Sheets project built to practise core data analysis skills.

## Skills Demonstrated
- SUMIF formulas to summarise data across sheets
- Absolute vs relative cell references
- Conditional formatting with custom formulas
- Dashboard design and data visualisation

## Project Structure
- **Transactions sheet** — raw data log of 30 transactions
- **Summary sheet** — formula-driven analysis by category
- **Dashboard sheet** — visual charts and KPIs
